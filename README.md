

## Informação geral
Bot do discord que atribui elo a jogadores e balanceia times.


	
## Dependências
Project is created with:

* Discord.js
* Mongoose
* Nodemon
* Dotenv
* Chalk

	
## Setup
Para rodar esse projeto, instale-o localmente usando o npm:

```
$ discord-balacingTeamsBot-node-mongodb
$ npm install
$ npm test

```
